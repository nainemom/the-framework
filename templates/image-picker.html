  • دریافت عکس در Cordova
  • قابلیت استفاده در المنت دلخواه

در Cordova و اندروید، انتخاب فایل به سادگی یک input از نوع فایل نیست. در آنجا باید از پلاگین‌های native استفاده کرد که خروجی آن‌ها با خروجی استاندارد browser متفاوت است. این دایرکتیو که به صورت اتریبیوت استفاده می‌شود، با توجه به پلتفرم مورد استفاده، از نوع مناسب انتخاب فایل استفاده می‌کند و بر روی خروجی آن توسط پرووایدر tfHttp چه اندروید باشد و چه غیره، بدون مشکل به سرور فرستاده می‌شود.

اتریبیوت‌های ng-model و ng-change هم همانطور که از نامشان پیداست، برای دریافت مدل و فانکشن هنگام تغییر استفاده می‌شود. 

//html
<button tf-image-picker ng-model="file" ng-change="onChange">
    Browse...
</button>

//js
file = ''; /* file object, browsed by user */
onChange = function(){
    alert('file changed!');
}
