قابلیت‌ها:

  • هماهنگی با فرم‌های bootstrap با استفاده از کلاس form-control
  • امکان انتخاب یک یا چند گزینه توسط کاربر.
  • امکان غیرفعال کردن جست‌وجو.
  • امکان اضافه کردن گزینه توسط کاربر.

این دایرکتیو از پلاگین selectize جی‌کوئری برگردانده شده است. در نتیجه تمام قابلیت‌هاش در اینجا نیز قابل استفاده است.

برای انتخاب چند گزینه از اتریبیوت multiple، برای غیرفعال کردن جست‌وجو از کلاس disable-search و برای امکان اضافه‌کردن آیتم توسط کاربر از اتریبیوت allow-create استفاده کنید. همچنین گزینه‌های از پیش تعیین شده، در قالب آرایه در جاوااسکریپت مقدار گرفته و به اتریبیوت options پاس داده می‌شوند. اتریبیوت ng-model هم به عنوان مدل این المنت استفاده می‌شود. 

//html
<tf-selectize class="form-control" allow-create multiple options="options" ng-model="input">
</tf-selectize>

//js
options = [
    { text: 'Option One', value: 1},
    { text: 'Option Two', value: 2},
    { text: 'Option Tree', value: 3}
];
input = ''; /* picked item value by user */
