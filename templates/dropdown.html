قابلیت‌ها:

  • هماهنگی با فرم‌های bootstrap با استفاده از کلاس form-control
  • امکان انتخاب یک یا چند گزینه توسط کاربر.
  • امکان حذف گزینه توسط کاربر.
  • امکان اضافه کردن گزینه توسط کاربر.

در نسخه قبلی، دایرکتیوی تحت پلاگین selectize در jquery نوشته شده بود. در این نسخه، این دایرکتیو تماما توسط angularjs و بدون وابستگی دیگر ایجاد شده است.

برای انتخاب چند گزینه از اتریبیوت multiple، برای امکان اضافه‌کردن آیتم توسط کاربر از اتریبیوت allow-create و برای اجازه حذف آیتم‌ها از allow-delete استفاده کنید و برای متن نمایشی قبل از انتخاب از placeholder استفاده کنید. همچنین گزینه‌های از پیش تعیین شده، در قالب آرایه در جاوااسکریپت مقدار گرفته و به اتریبیوت options پاس داده می‌شوند. اتریبیوت ng-model هم به عنوان مدل این المنت استفاده می‌شود. 

//html
<tf-select class="form-control" allow-create multiple options="options" ng-model="input">
</tf-select>

//js
options = [
    { text: 'Option One', value: 1},
    { text: 'Option Two', value: 2},
    { text: 'Option Tree', value: 3}
];
input = []; /* picked item value by user */
