  • امکان قراردادن متن دل‌بخواه.

برای دریافت ایونت و متن درخواستی کاربر برای سرچ، می‌توانید از اتریبیوت search و برای تایین متن دل‌بخواه برای placeholder از اتریبیوتی به همین نام استفاده کنید.

توجه داشته باشید هنگامی که فانکشن ورودی اتریبیوت search را وارد می‌کنید، آرگومان متن ورودی کاربر را تحت عنوان $text به فانکشن پاس دهید! 

//html
<tf-searchbar open="searchbar" placeholder="{‌{placeholder}‌}" search="startSearch($text)">
</tf-searchbar>

//js
startSearch = function(searchText){
    alert( 'start searching for '+ searchText)
}
searchbar = false; /* show = true, hide = false. */
