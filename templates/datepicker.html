قابلیت‌ها:

  • امکان استفاده از تقویم میلادی و شمسی.
  • هماهنگی با کلاس form-control

نوع تقویم را می‌توانید در اتریبیوت calendar مشخص کنید. تقویم‌های قابل استفاده:

  • jalaali: تقویم جلالی (حالت پیش‌فرض)
  • gregorian: تقویم میلادی

همچنین می‌توانید برای استفاده در فرم‌های بوت‌استریپ از کلاس form-control استفاده کنید.

اتریبیوت‌های ng-model و ng-change هم همانطور که از نامشان پیداست، برای دریافت مدل و فانکشن هنگام تغییر استفاده می‌شود. 

//html
<tf-datepicker class="form-control" ng-model="date" ng-change="onChange"  calendar="{‌{calendar}‌}">
</tf-datepicker>

//js
date = ''; /* picked date by user */
onChange = function(){
    alert('date changed!');
}
calendar = 'jalaali'; /* or 'gregorian' */
