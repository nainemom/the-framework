قابلیت‌ها:

  • ارسال فایل.
  • ارسال محتوا بوسیله FormData.
  • مقداردهی content-type به صورت اتوماتیک.

برای استفاده از امکانات این پرووایدر، باید tfHttp را به کنترلر مورد نظر پاس دهید. متود‌های قابل استفاده عبارت‌اند از:

  • get(url, [data])
    ارسال به وسیله متود get (پارامتر اول آدرس صفحه و پارامتر دوم دیتای ارسالی)
  • post(url, [data]) ارسال به وسیله متود post (پارامتر اول آدرس صفحه و پارامتر دوم دیتای ارسالی)
  • serialize(obj): برای تبدیل یک آبجکت به querystring

مقادیری که در فانکشن config قابل مقداردهی هستند، عبارت‌اند از:

  • address: اگر آدرس سرور شما در تمام ارسال‌ها یکی است، می‌توانید این پارامتر را مقداردهی کرده و دیگر در ارسال‌ها مقدار url را ذکر نکنید.
  • sid: مخفف session id بوده و اگر مقداردهی شود، پارامتر _sid به همراه تمام ارسال‌ها فرستاده می‌شود. مناسب برای اپلیکیشن‌های cordova که در آنها cookie وجود ندارد!

بعد از ارسال، پاسخ دریافتی از سرور به متود then و در صورت بروز اشکال، مقدار error به متود catch ارسال می‌شود. 

//js
function Controller($tfHttp){
    var data = {
        sort: 'stars',
        order: 'desc',
        q: 'angular',
    } // example data
    var url = "https://api.github.com/search/repositories"; // example url
    $tfHttp.get(url, data).then(function(response) {
        // success
    }).catch(function(error){
        // error
    });
}
