توضیحات اینجا قرار می‌گیرد.

قابلیت‌ها:

  • تعیین مکان محتوا به وسیله flexbox
  • امکان قرار دادن دکمه
  • امکان قرار دادن توضیحات
  • امکان قرار دادن لوگو
  • امکان اضافه و کم کردن بخش‌ها

هر section قابلیت حذف‌شدن دارد. همچنین کلاس‌های قابل استفاده برای هر بخش عبارت‌اند از:

  • icon: آیکن سایز بزرگ برای لوگو
  • title: نوشته عنوان
  • icon button: دکمه آیکن. مناسب برای سایدبار و...
  • icon text: دکمه نوشته. مناسب برای فعالیت‌های خاص.

در داخل هر section که کلاس icon دارد، می‌توانید از یک تگ i و کلاس آیکن‌های font-awesome استفاده کنید. همچنین در تگ عنوان که با کلاس title مشخص می‌شود، می‌توانید از دو تگ h4 برای عنوان اصلی و p برای توضیحات استفاده کنید. استفاده از تگ p اختیاری است. 

//html
<nav class="tf-navbar">
    <section class="icon">
        <i class="fa fa-twitter"></i>
    </section>
    <section class="title">
        <h4>عنوان</h4>
        <p>توضیحات</p>
    </section>
    <section class="icon button">
        <i class="fa fa-search"></i>
    </section>
    <section class="icon button">
        <i class="fa fa-bars"></i>
    </section>
</nav>
